# Aura Tienda Holística  
**Sanación Emocional · Equilibrio Energético · Bienestar Consciente**

Este proyecto es la base del sitio web oficial de **Aura Tienda Holística**, un espacio digital diseñado para ofrecer productos y servicios terapéuticos enfocados en:

### Servicios
- Liberación Emocional (EFT / Tapping)
- Biomagnetismo Medicinal (a distancia)
- Mesa Cuántica Energética

### Productos
- Baños de sales  
- Kits holísticos  
- Pulseras energéticas  
- Sahumerios  
- Sprays energéticos  
- Velas terapéuticas  

El sitio está concebido como un **santuario digital**, donde la experiencia, la confianza y la coherencia energética son más importantes que la venta agresiva.

Idioma principal: **Español**  
Idiomas secundarios: **Configurables (inglés y otros a futuro)**  
Detección automática por navegador con opción de cambio manual.

---

## Propósito

Aura no es solo una tienda online.  
Es un espacio de acompañamiento para personas que buscan:

- Sanación emocional.
- Equilibrio energético.
- Comprensión consciente de los productos que usan.
- Acceso a terapias con profesionales reales.

No existen dashboards, métricas personales ni sistemas de seguimiento.  
El foco está en la experiencia, la claridad y el contacto humano.

---

## Tecnología

- Node.js  
- Framework moderno (React / Next.js)  
- Asistente de IA para orientación (no automatización fría)  
- Sistema de reservas seguro  
- Arquitectura multilenguaje  
- Interfaz oscura por defecto, eficiente y calmante
